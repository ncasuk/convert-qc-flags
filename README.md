-----

## Table of Contents

## Installation

```console
pip install convert-qc-flags
```

## Usage

```python
import convert_qc_flags
convert_qc_flags.convert_qc_flag_attributes(<name-of-file>)
```
`<name-of-file>` is converted in place.
